# Frontend Reusable Components

## Purpose
Holds all global, stateless, and state-independent reusable React UI widgets.

## Files
- `Button.jsx`: General actions button supporting loads.
- `Card.jsx`: Glassmorphism dashboard container cards.
- `Sidebar.jsx`: Unified dashboard sidebar routing interface.
- `Navbar.jsx`: Top menu and branding bar.
- `LoadingSkeleton.jsx`: Pulsing page loaders.
- `Toast.jsx`: Standard popup notices.
- `Modal.jsx`: Centered dynamic overlay container.
- `ChartsPlaceholder.jsx`: SVG-based simulated graphical displays.

## Responsibilities
- Provide styling consistency.
- Decouple specific page logic from common structures.
