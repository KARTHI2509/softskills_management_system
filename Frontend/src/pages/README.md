# Frontend Pages

## Purpose
Contains the main route views. These components represent whole screens or sub-screens within the application.

## Files
Includes Landing, About, Contact, Login, Register, ForgotPassword, ResetPassword, plus:
- `StudentDashboard.jsx` / `FacultyDashboard.jsx` / `PlacementDashboard.jsx` / `AdminDashboard.jsx`
- `CommunicationModule.jsx`, `MockInterview.jsx`, `GroupDiscussion.jsx`, `ResumeBuilder.jsx`, `Aptitude.jsx`, `Leaderboard.jsx`
- `Reports.jsx`, `Notifications.jsx`, `Settings.jsx`, `Profile.jsx`, `NotFound.jsx`

## Responsibilities
- Mount layouts.
- Dispatch data calls to service wrappers.
- Render UI pages and grids.
