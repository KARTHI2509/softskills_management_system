# Frontend Layouts

## Purpose
Visual shell structures that wrap different sets of pages.

## Files
- `MainLayout.jsx`: Used for landing, about, contact pages, showing navigation header and simple footers.
- `DashboardLayout.jsx`: Used for internal routes (student, faculty, placement, admin), containing the responsive sidebar navigation layout.
- `AuthLayout.jsx`: Layout container shell for user registration, log in, password resets.

## Responsibilities
- Control visual layout structures.
- Group common elements like headers, sidebar widgets, footer contents.
