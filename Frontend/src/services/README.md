# Frontend Services

## Purpose
Exposes modular API request methods mapping variables to client configurations.

## Files
- `authService.js`: Registers and logs in users, manages local storage token sets.
- `studentService.js`: Pulls dashboard states and graphs.
- `facultyService.js`: Routes evaluations and assigns.
- `placementService.js`: Obtains eligibility listings and analytics.

## Responsibilities
- Abstract client-side endpoint calling logic.
- Standardize payload formatting before server handoff.
